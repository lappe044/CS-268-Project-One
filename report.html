1. You will create a website for a client of your choosing. Your client could be a family member, a business, a student group, a friend, an online community, and so on. The only restriction is that the client cannot be just you.
    • Our website was created for the client Epic MT, a backpacking company whose mission is to make backpacking more accessible to everyone.
  2. Your site's webpages must include some sort of headers.
    • Our website includes a consistent header on everypage. The header includes a cool background picture, the name of our website (which acts as a link to the homepage), and sits above the navbar.
  3. Your site must contain at least ten separate pages, each having a meaningful purpose.
    • Our website contains ten separate pages (a homepage, contact page, about page, and a page for each of the 7 continents). Each page has a meaningful purpose. The homepage introduces the company, the about page allows visiters to learn more about the company, the contact page allows visiters to inquire about booking trips, and each of the continent pages details the trips available in that continent.
  4. Your site's webpages must include some sort of navigation links.
    • Our webpage includes navigation links in the navbar, allowing visiters to easily navigate around the webpage. Additionally, the webpage includes external links in the footer to social media platforms.
  5. Your site must support distinct layouts for large and small screens. For instance, multicolumn layouts on large screens should be reduced to single column layouts on small screens. You are encouraged to implement separate layouts using CSS media queries.
    • Our website supports distinct layouts for large and small screens using media queries and flexboxes. For example, our pictures and font shrink to accomadate the smaller size of a mobile screen and our flexbox layouts (in about page and on homepage) change from a 4 entry layout for large screens to a two entry layout for small screens.
  6. Your site must maintain a coherent style across its pages. Global style rules must be placed in a shared stylesheet. Individual pages’ style rules must be placed in a separate stylesheets.
    • Our site maintains coherent style across its pages. Global style rules are placed in a shared stylesheet entitled global whiel individual page's style rules are in separate stylesheets that match the description of the page.
  7. Your site must include images, and their alt attributes must adequately describe the image content to users of screen readers, user with images disabled, and webcrawlers.
    • The site inclues images and each image has an alt attribute describing the content
  8. Each page must declare its favicon.
    • Each page includes a mountain favicon
  9. Your site’s non-HTML resources must be organized. Images must be placed in a directory named images. CSS files must be placed in a directory named css.
    • Our website oragnizes non-HTML resources in folders. Images are placed in a directory named images and CSS files are placed in a directory named css.
  10. Your site must not be built using any libraries or frameworks. Do not use Bootstrap, jQuery, React, Angular, Vue, or any other third-party tools. You may, however, reimplement the ideas you see in these third-party tools.
    • Our website does not use any libraries or frameworks.
  11. Each page and stylesheet of your site must validate without any warnings or errors using the W3C’s HTML validator (Links to an external site.) and CSS validator (Links to an external site.).
    • Each page and stylesheet validates without any warnings or errors using the validation sites.
  12. Your site could be stored in a Git repository that you have shared with your instructor. On GitHub, GitLab, and Bitbucket, share with the user gomesr-uwec.(Not required for this project but recommended)
    • Our site is stored in a Git repository
  13. The names of your files should be entirely lowercase and contain only alphanumeric characters.
    • The names of our files are entirely lowercase and contain only alphanumeric characters.

    • Our content is neatly aligned.
